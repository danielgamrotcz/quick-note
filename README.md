# Capto

macOS app for quick notes to Notion. Global shortcut brings up a floating panel, you type a note, hit Enter — it's in your Notion database.

## Features

- **Global shortcut** — Ctrl+Opt+Cmd+I opens Capto from anywhere
- **Floating panel** — borderless, always-on-top, disappears after submission
- **Notion integration** — notes go straight to your Notion database via API
- **Customizable shortcut** — change the default hotkey in settings

## Requirements

- macOS 14+
- Intel & Apple Silicon
- Notion account + API integration
